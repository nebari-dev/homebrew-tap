# homebrew-tap

Homebrew formulae and casks for nebari-dev tools.

```bash
brew tap nebari-dev/tap
```

## Available packages

### nic (Nebari Infrastructure Core)

```bash
brew install nebari-dev/tap/nic
```

Cloud infrastructure management for Nebari. The `nic` cask is published
automatically by the [nebari-infrastructure-core](https://github.com/nebari-dev/nebari-infrastructure-core)
release pipeline (GoReleaser).

> Homebrew casks install on macOS. On Linux, download the release tarball from
> the [releases page](https://github.com/nebari-dev/nebari-infrastructure-core/releases)
> instead.

### skillsctl

```bash
brew install skillsctl
```

CLI for discovering, installing, and publishing Claude Code skills.
